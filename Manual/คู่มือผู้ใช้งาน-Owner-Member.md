## 3. หน้า Projects — จัดการโครงการ

คลิกเมนู **Projects** ที่ Sidebar ด้านซ้าย

### 3.1 Stat Cards (ด้านบน)

แสดงจำนวนโครงการแยกตามสถานะ:
- **Total** — จำนวนโครงการทั้งหมด
- **Active** — กำลังดำเนินการ (เขียว)
- **Delay** — ล่าช้า (แดง)
- **Completed** — เสร็จแล้ว (ฟ้า)
- **Hold** — พักไว้ (ส้ม)
- **Cancelled** — ยกเลิก (เทา)

### 3.2 การค้นหาและกรอง

| เครื่องมือ | วิธีใช้ |
|-----------|---------|
| **ช่องค้นหา** | พิมพ์ชื่อโครงการ ระบบจะกรองอัตโนมัติ |
| **ตัวกรองสถานะ** | ติ๊ก Active / Delay / Completed / Hold / Cancelled เพื่อแสดงเฉพาะสถานะที่ต้องการ |
| **เรียงลำดับ** | คลิกปุ่ม Sort เลือก: ชื่อ A→Z, Z→A, ใหม่สุด, เก่าสุด |

### 3.3 มุมมอง 3 แบบ

ปุ่มสลับมุมมองอยู่ด้านขวาบน:

#### มุมมอง Card (ตาราง)
- แสดงโครงการเป็นการ์ด 3 คอลัมน์
- แต่ละการ์ดแสดง: ชื่อ, คำอธิบาย, สถานะ, Progress Bar, จำนวนสมาชิก
- **คลิกการ์ด** → ไปหน้ารายละเอียดโครงการ
- **คลิกจุด 3 จุด (⋮)** → เมนู: ดูโครงการ / แก้ไข / ลบ

#### มุมมอง Board (Kanban)
- แสดงเป็นกระดาน 5 คอลัมน์ตามสถานะ
- **ลากการ์ด** จากคอลัมน์หนึ่งไปอีกคอลัมน์ → เปลี่ยนสถานะโครงการอัตโนมัติ
- **คลิกการ์ด** → ไปหน้ารายละเอียดโครงการ
- **คลิกจุด 3 จุด (⋮)** → เมนู: ดูโครงการ / แก้ไข / ลบ

#### มุมมอง List (รายการ)
- แสดงเป็นรายการแถวเดียว พร้อมข้อมูลครบ
- **คลิกแถว** → ไปหน้ารายละเอียดโครงการ

### 3.4 สร้างโครงการใหม่

1. คลิกปุ่ม **+ New Project** (มุมขวาบน)
2. กรอกข้อมูล:

| ช่อง | บังคับ | รายละเอียด |
|------|--------|-----------|
| Project Name | ใช่ | ชื่อโครงการ (ไม่เกิน 100 ตัวอักษร) |
| Description | ไม่ | คำอธิบายโครงการ (ไม่เกิน 500 ตัวอักษร) |
| Color | ไม่ | เลือกสีของโครงการ (8 สีให้เลือก) |
| Status | ไม่ | สถานะเริ่มต้น (ค่าเริ่มต้น: Active) |
| Start Date | ไม่ | วันที่เริ่มต้นโครงการ |
| End Date | ไม่ | วันที่สิ้นสุดโครงการ |

3. คลิก **Create** เพื่อสร้าง

### 3.5 แก้ไขโครงการ

1. คลิกจุด 3 จุด (⋮) บนการ์ดโครงการ
2. เลือก **Edit Project**
3. แก้ไขข้อมูลที่ต้องการ
4. คลิก **Save**

### 3.6 ลบโครงการ

1. คลิกจุด 3 จุด (⋮) บนการ์ดโครงการ
2. เลือก **Delete Project**
3. ระบบจะถามยืนยัน → คลิก **Delete**

> **คำเตือน:** การลบโครงการจะลบงาน (Task) ทั้งหมดในโครงการด้วย ไม่สามารถกู้คืนได้

## 4. หน้า Project Detail — รายละเอียดโครงการ

คลิกชื่อโครงการจากหน้า Projects เพื่อเข้าสู่หน้ารายละเอียด

### 4.1 ส่วน Header

- **Breadcrumb** — เส้นทาง: Projects > ชื่อโครงการ (คลิกเพื่อกลับ)
- **ชื่อโครงการ** พร้อมคำอธิบาย
- **เส้นสีด้านบน** — แสดงสีตามสถานะโครงการ

### 4.2 Timeline Cards (ถ้ามีวันที่กำหนด)

แสดง 3 การ์ด:
- **Project Start** — วันเริ่มต้นโครงการ
- **Target Finish** — วันที่กำหนดเสร็จ
- **Time Remaining** — จำนวนวันที่เหลือ

### 4.3 Stat Cards

| การ์ด | รายละเอียด |
|-------|-----------|
| Total Tasks | จำนวน Task ทั้งหมด |
| Completed | จำนวน Task ที่เสร็จแล้ว |
| In Progress | จำนวน Task ที่กำลังทำ |
| Progress | % ความคืบหน้ารวม |

### 4.4 Board View (Kanban)

กระดานงาน 5 คอลัมน์:
- **To Do** — งานที่ยังไม่เริ่ม
- **In Progress** — งานที่กำลังทำ
- **Done** — งานที่เสร็จแล้ว
- **Hold** — งานที่พักไว้
- **Cancelled** — งานที่ยกเลิก

**การใช้งาน:**
- **ลากการ์ด** ระหว่างคอลัมน์ → เปลี่ยนสถานะ Task
- **คลิกการ์ด** → เปิดหน้ารายละเอียด Task
- **คลิกจุด 3 จุด (⋮)** → เมนู: ดู / แก้ไข / ลบ

**แต่ละการ์ดแสดง:**
- ชื่อโครงการ (ด้านบน)
- ชื่องาน (ตัวหนา)
- ลำดับความสำคัญ (Priority Badge สี)
- วันกำหนดส่ง + ป้าย "Delay" หรือ "Due soon" (ถ้ามี)
- ชื่อผู้รับผิดชอบ (Assignees)

### 4.5 สร้าง Task ใหม่

1. คลิกปุ่ม **+ New Task**
2. กรอกข้อมูล:

| ช่อง | บังคับ | รายละเอียด |
|------|--------|-----------|
| Title | ใช่ | ชื่องาน (ไม่เกิน 200 ตัวอักษร) |
| Description | ไม่ | คำอธิบายงาน |
| Priority | ไม่ | ลำดับความสำคัญ (ค่าเริ่มต้น: Medium) |
| Status | ไม่ | สถานะเริ่มต้น (ค่าเริ่มต้น: To Do) |
| Assignees | ไม่ | เลือกผู้รับผิดชอบ (เลือกได้หลายคน) |
| Start Date | ไม่ | วันที่เริ่มต้น |
| Due Date | ไม่ | วันกำหนดส่ง |

3. คลิก **Create**

### 4.6 ปุ่ม Export

- **Export Excel** — ดาวน์โหลดข้อมูล Task เป็นไฟล์ Excel
- **Save PDF** — บันทึกเป็นไฟล์ PDF

## 5. หน้า My Tasks — งานของฉัน

คลิกเมนู **My Tasks** ที่ Sidebar แสดงเฉพาะงานที่ได้รับมอบหมาย หรือสร้างเอง

### 5.1 Stat Cards

แสดงจำนวน Task แยกตามสถานะ: Total, To Do, In Progress, Done, Hold, Cancelled

### 5.2 กรองตามลำดับความสำคัญ

ติ๊กเลือก: **Urgent** / **High** / **Medium** / **Low** เพื่อแสดงเฉพาะ Priority ที่ต้องการ

**การใช้งาน:**

- **คลิกจุด 3 จุด (⋮)** → View/Edit หรือ Delete Task

### 5.4 สร้าง Task ใหม่

1. คลิกปุ่ม **+ New Task**
2. เลือกโครงการ (Project) จาก dropdown
3. กรอกชื่องาน, คำอธิบาย, Priority, Status, Due Date
4. คลิก **Create**

## 6. การดูและแก้ไข Task (Task Detail)

เมื่อคลิกการ์ด Task จะเปิดหน้าต่าง Modal แสดงรายละเอียดงาน

### 6.1 ข้อมูลที่แสดง

| ส่วน | รายละเอียด |
|------|-----------|
| **ชื่องาน** | ชื่อ Task (ตัวหนาใหญ่) |
| **คำอธิบาย** | รายละเอียดงาน |
| **Metadata** | Assignees, Start Date, Due Date, Priority, Status, Created By, Progress |
| **Sub-tasks** | รายการงานย่อย พร้อม checkbox |
| **Comments** | ความคิดเห็น + ไฟล์แนบรูปภาพ |
| **Daily Updates** | ประวัติการอัปเดตรายวัน |

### 6.2 การแก้ไข Task

1. คลิกปุ่ม **Edit** (ไอคอนดินสอ)
2. แก้ไขข้อมูลได้ทั้งหมด:
   - ชื่อ, คำอธิบาย
   - Priority (Urgent / High / Medium / Low)
   - Status (To Do / In Progress / Done / Hold / Cancelled)
   - Assignees (เพิ่ม/ลบผู้รับผิดชอบ — เลือกได้หลายคน)
   - Start Date / Due Date
3. คลิก **Save** เพื่อบันทึก หรือ **Cancel** เพื่อยกเลิก

### 6.3 Sub-tasks (งานย่อย)

**Tab "Sub-tasks":**
1. พิมพ์ชื่องานย่อยในช่อง Input
2. คลิก **Add** หรือกด Enter
3. ติ๊ก checkbox เพื่อ mark ว่าเสร็จ
4. คลิกไอคอนถังขยะเพื่อลบ

### 6.4 Comments (ความคิดเห็น)

**Tab "Comments":**
1. พิมพ์ข้อความในช่อง
2. (ถ้าต้องการ) คลิกไอคอนรูปภาพเพื่อแนบไฟล์
3. คลิก **Send** หรือกด Enter
4. ความคิดเห็นจะแสดงพร้อมชื่อผู้เขียน และเวลา

### 6.5 Daily Updates (อัปเดตรายวัน)

**Tab "Daily Updates":**
1. คลิก **+ Add Update**
2. กรอก:
   - หมายเหตุ (สิ่งที่ทำวันนี้)
   - Progress % (ความคืบหน้า)
   - สถานะใหม่ (ถ้าต้องการเปลี่ยน)
3. คลิก **Submit**
4. ประวัติจะแสดงเป็น Timeline ด้านล่าง

### 6.6 อัปเดต Progress

- ใน Task Detail สามารถปรับ **Progress (0-100%)** ได้
- เมื่อลาก Task ไปคอลัมน์ **Done** → Progress จะถูกตั้งเป็น **100%** อัตโนมัติ

## 10. คำอธิบายสถานะและลำดับความสำคัญ

### 10.1 สถานะโครงการ (Project Status)

| สถานะ | สี | ความหมาย |
|-------|----|----------|
| **Active** | เขียว | กำลังดำเนินการ |
| **Delay** | แดง | ล่าช้ากว่ากำหนด |
| **Completed** | ฟ้า | เสร็จสิ้นแล้ว |
| **Hold** | ส้ม | พักไว้ชั่วคราว |
| **Cancelled** | เทา | ยกเลิก |

### 10.2 สถานะงาน (Task Status)

| สถานะ | สี | ความหมาย |
|-------|----|----------|
| **To Do** | เทา | ยังไม่เริ่ม |
| **In Progress** | ฟ้า | กำลังดำเนินการ |
| **Done** | เขียว | เสร็จแล้ว (Progress 100%) |
| **Hold** | ส้ม | พักไว้ชั่วคราว |
| **Cancelled** | เทา | ยกเลิก |

### 10.3 ลำดับความสำคัญ (Priority)

| Priority | สี | ความหมาย |
|----------|----|----------|
| **Urgent** | แดง | เร่งด่วนมาก ต้องทำทันที |
| **High** | ส้ม | สำคัญมาก |
| **Medium** | เหลือง | ปานกลาง (ค่าเริ่มต้น) |
| **Low** | เขียว | ต่ำ ทำเมื่อมีเวลา |

## เคล็ดลับการใช้งาน

1. **Drag & Drop** — ลากการ์ดระหว่างคอลัมน์เพื่อเปลี่ยนสถานะได้ทันที ไม่ต้องเปิดแก้ไข
2. **หลายผู้รับผิดชอบ** — Task หนึ่งสามารถมีผู้รับผิดชอบได้หลายคน เลือกจาก dropdown แบบ multi-select
3. **Daily Update** — แนะนำให้อัปเดต progress ทุกวัน เพื่อให้ทีมเห็นความคืบหน้า
4. **ค้นหาเร็ว** — ใช้ช่องค้นหาในหน้า Projects เพื่อหาโครงการที่ต้องการ
5. **ป้าย Delay / Due soon** — ระบบจะแสดงป้ายเตือนอัตโนมัติเมื่อ Task ใกล้กำหนดหรือเลยกำหนด
